สร้าง QR Code ที่สแกนแล้วแสดงรูปสุ่ม 3D

📱 สร้าง QR Code

🖼️ จัดการรูปภาพ

วิธีเพิ่มรูป:
1. แก้ไขไฟล์ config.js ใน repo
2. ใส่ URL ของรูปภาพในอาเรย์ images
3. Push ขึ้น GitHub แล้วรอ 1-2 นาที

แหล่งรูปที่ใช้ได้:
• อัปรูปเข้า repo โฟลเดอร์ images/ → ใช้ path เช่น "images/photo1.jpg"
• ใช้ URL รูปจาก Imgur, Cloudinary, Firebase Storage
• ใช้รูปตัวอย่างจาก picsum.photos

* ข้อมูลจาก localStorage ของ browser นี้เท่านั้น
